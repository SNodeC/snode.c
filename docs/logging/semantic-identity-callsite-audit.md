# Semantic identity call-site audit

## Scope

This PR remains audit-only. It changes no production logging call sites and does not change logging behavior, semantic formatting, colors, JSON, backend routing, LogRecord fields, LogManager policy, CLI/config, macros, helper design, install rules, or application code.

## Methodology

Raw `rg` matches are evidence, not call sites. Each raw match was first sorted into `PRODUCTION_LOGGING`, `APPLICATION_DEMO_LOGGING`, `NON_LOGGING`, `TEST`, or `DOCUMENTATION`. Only actual identity-bearing logging expressions then receive cleanup decisions (`REMOVE_LATER`, `KEEP`, `FOLLOW_UP_SCOPED_HELPER_DESIGN`, `LEGACY_OR_USER_FACING`, or `UNKNOWN_NEEDS_REVIEW`). Scope construction, declarations, definitions, callback captures that do not themselves emit a log record, OpenSSL ex-data, request/response storage, event receiver names, and local variables are tracked as non-logging evidence rather than cleanup candidates. Multiline stream expressions were inspected by owner and complete logging expression; continuation lines inherit surfaces such as `sysError(coreSocketLog())` instead of being classified from an isolated `<<` line.

## Search commands

```sh
rg -n "getConnectionName|getInstanceName" src tests/unit/log docs/logging
rg -n "<<.*get(Connection|Instance)Name\(" src tests/unit/log
rg -n "get(Connection|Instance)Name\(\).*<<" src tests/unit/log
rg -n "\.\s*(trace|debug|info|warn|error|critical)\s*\([^\n]*get(Connection|Instance)Name" src tests/unit/log
rg -n "(frameworkLog|\blog\(|appLog|expressLog|httpLog|httpClientLog|httpClientEventSourceLog|webSocketLog|webSocketSubProtocolLog|mqttLog|mqttWebSocketLog|mqttBrokerLog|mqttSessionLog|coreSocketLog|tlsConfigLog).*get(Connection|Instance)Name" src tests/unit/log
rg -n "(getSocketConnection\(\)->getConnectionName|getSocketConnection\(\)->getInstanceName|socketConnection->getConnectionName|socketConnection->getInstanceName|config->getInstanceName|Super::getConnectionName|Super::getInstanceName|subProtocolContext->getSocketConnection\(\)->getConnectionName)" src tests/unit/log
```

The first command is reported below as 646 raw matches after excluding this report file from self-matches during recounting.
